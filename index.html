// 基本の使い方

1
小説家になろうなどの目次URLをコピーして「読む」タブに貼り付ける
例: https://ncode.syosetu.com/n1234ab/
2
「読込」を押すと話一覧が表示される。読みたい話をタップ
3
自動で読み上げ開始。話の終わりに5秒カウント後、次の話へ自動進行

// しおり・履歴

  • 読み上げ中は10段落ごとに自動でしおり保存
  • 次回同じURLを入力すると「🔖 読みかけ」バナーが出る
  • 「履歴」タブから過去に読んだ小説を再開できる

// 再生コントロール

  • ⏮ 前の段落 ／ ⏭ 次の段落
  • ■ 停止（もう一度▶で最初から）
  • 次→ 今すぐ次の話へ進む
  • 速度・ピッチは − ＋ で0.1ずつ調整

// バックグラウンド再生（iPhone）

Safariのまま画面を閉じると止まる場合があります。対策：

  • 設定 → 画面表示と明るさ → 自動ロックをオフ
  • 止まったら画面をタップすると再開
  • ホーム画面に追加（共有→ホーム画面に追加）すると安定

// Google TTS 高品質音声（任意）

PCやAndroidで取得してiPhoneに貼るだけでOKです

1
Google Cloud Consoleを開く
ブラウザで下記にアクセスしGoogleアカウントでログイン
console.cloud.google.com
2
プロジェクトを作成
画面上部「プロジェクトを選択」→「新しいプロジェクト」→ 名前は何でもOK（例: novel-reader）→「作成」
3
Text-to-Speech APIを有効化
左メニュー「APIとサービス」→「ライブラリ」→ 検索欄に text to speech と入力 →「Cloud Text-to-Speech API」→「有効にする」
4
請求先を設定（無料利用でも必要）
クレジットカードを登録しますが 月100万文字まで無料 です。このアプリは90万文字で警告・100万文字で自動停止するので通常は課金されません
5
APIキーを発行
「APIとサービス」→「認証情報」→「＋認証情報を作成」→「APIキー」
→ AIzaSy… で始まるキーが表示される → コピー
6
（推奨）キーを制限
「キーを制限する」→「APIの制限」→「Cloud Text-to-Speech API」のみ選択 → 保存。他のAPIに使われなくなり安全
7
このアプリに貼り付け
「設定」タブ → APIキー欄にペースト →「保存して使う」→ 完了！
🔒 キーはこの端末にのみ保存。外部サーバーには送信しません。
📊 使用量は「設定」タブで確認できます。
💰 90万文字で警告 / 100万文字でKyokoに自動切替（課金防止）
🔄 翌月1日に自動リセット。課金プランの場合は手動リセット可
